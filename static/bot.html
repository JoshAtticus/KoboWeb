KoboWeb Proxy Bot

This page identifies the KoboWeb proxy bot and explains what the requests are.

KoboWeb is a lightweight web client designed for e-readers (like Kobos and Kindles) and retro devices (old iOS, Android, Game Consoles etc).

What you'll see in logs

Requests from the proxy use this user agent:

KoboWeb/1.0 (+https://kobo.joshattic.us/bot)

Abuse prevention

We implement strict rate limiting and domain allowlisting to reduce abuse and accidental scraping of other sites.
